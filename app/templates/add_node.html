Prerequisites

Before adding a node, ensure:

  • The target machine is running Ubuntu 20.04+ or similar Linux distribution
  • SSH access is available (password or existing key authentication)
  • The SSH user has sudo privileges
  • The machine has internet connectivity for downloading packages
  • Ports 22 (SSH), 16443 (Kubernetes API), and 10250-10259 (Kubelet) are accessible

Note: After adding the node, you will receive:

  • A unique SSH key pair generated automatically
  • Step-by-step instructions including SSH server configuration
  • Security hardening steps to disable password authentication
  • Connection testing to verify the setup
SSH Server Configuration Required: The setup instructions will include steps to configure the SSH server on the target node for secure key-based authentication, including disabling password authentication for enhanced security.
