Monitoring Information
How Power Monitoring Works
  • Continuously monitors UPS status every {{ monitoring_status.monitoring_interval }} seconds
  • Checks for power events (power loss, low battery, etc.)
  • Executes configured cluster actions when events occur
  • Logs all actions and results for audit purposes
Power Events Monitored
  • Power Loss: UPS switches to battery power
  • Low Battery: Battery charge below threshold
  • Critical Battery: Battery critically low (10%)
  • Power Restored: Main power returns

System Requirements
Prerequisites for Power Monitoring
  • NUT Packages: Install with sudo apt install nut nut-client nut-server nut-driver
  • UPS Connection: UPS must be connected via USB and detected by the system
  • NUT Configuration: UPS must be configured in NUT with proper drivers
  • Service Permissions: NUT services must be running and accessible
  • Power Rules: At least one power management rule must be configured
